# OmniRemote™ Brand Guidelines

© 2026 One Eye Enterprises LLC

## Trademark Assets

- **Word Mark**: OmniRemote (one word, bold, with cyan underline accent)
- **Logo**: Purple remote with cyan accents and signal waves
- **Tagline**: "Control everything. Learn anything. Use any remote."
- **Secondary Tagline**: "One Remote to Rule Them All™"

## Brand Colors

| Color | Hex Code | Usage |
|-------|----------|-------|
| **Primary Purple** | `#7C3AED` | Main brand color, buttons, active states |
| **Accent Cyan** | `#2DD4BF` | Signal accents, highlights, underlines |
| **Dark Navy** | `#0f172a` | Backgrounds |
| **Card Background** | `#1a1a2e` | Cards, sidebar |
| **Border** | `#2a2a4a` | Borders, dividers |
| **Text Primary** | `#e8e8e8` | Main text |
| **Text Secondary** | `#888888` | Muted text |

## Typography

- **Wordmark**: Bold, 700 weight, with cyan underline
- **Body**: System UI / Sans-serif
- **Code**: Monospace

## Logo Usage

The logo should always appear with:
- Purple gradient body (#7C3AED to #8B5CF6)
- Cyan accent elements (#2DD4BF)
- Signal waves on left and right sides
- Dark background (#0f172a or similar)

## Do NOT

- Split "OmniRemote" into two colors
- Use different accent colors
- Remove the cyan underline from the wordmark
- Alter the tagline wording
